## 安装

使用本框架前，请先设置css:
```
*,*::befoer,*::after{ box-sizing:border-box; }
```
你还需要设置默认颜色等变量
```
html{
  --button-height:32px;
  --button-bg:white;
  --font-size:14px;
  --button-active-bg:#eee;
  --border-radius:4px;
  --color:#333;
  --border-color:#999;
  --border-color-hover:#666;
  }
```

IE 8 及以上支持。
